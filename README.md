# Sustainable computing in the research lifecycle

This repository contains the slides for a talk given as part of the [NetDRIVE Summer School 2026]().
The aim of this talk is to give a brief overview of the key ways that computing effects the environment,
principles for reducing these environmental impacts,
and to describe how these principles can be applied at different stages of a research project.
